Web Security

In today's world, having a basic understanding of web security is a must. There are simple ways to help protect yourself while online. A few ways to do so:

  • Install Anti-Virus Software
  • Update Passwords Often
  • Visit Trusted Sites
  • Download Only From Trusted Sites
  • Keep Operating System Up To Date

You also may be surprised as to what a web app can "see" information wise. The Navigator Object is used to locate information about the current browser window. It can be used with all browsers. The following properties of the Navigator Object gathers the following information:

  • appName: Name of web browser
  • appVersion: Version of the web browser
  • geolocation: Access to the user's location
  • onLine: If the browser is connected to the internet
  • platform: Operating system current CPU is using
  • userAgent: Information stored including browser, platform name, and compatibility
